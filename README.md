# **Ubuntu Desnapper Customizable Ver**
**This is a fork of thvret77's Ubuntu desnapper. I am learning C so I wanted to try this.**

**This Tool Allows You To Remove The Snaps And Install Suggested Software With Little To No Effort**

**This Tool Is Open Source, If You're Ever Afraid Of Running It, You Can Build From Source**

**Once again, this is a first project. If You Find A Bug, Please Report It.**

This Utility requires Root Privileges.

- [x] It Removes Snaps from Ubuntu
- [x] it Installs Recommended Software That You Can Choose From (Optional)
- [x] It Reinstalls Firefox Using apt
- [ ] It Has Graphical User Interface

> Note: This Tool Will Be Regularly Updated And It Will Provide A Clean Experience For New Users Consider Contributing To This Project!



> [!TIP]
> To Run The Executable Open The Terminal And Type "./Desnap-Linux-X64"
